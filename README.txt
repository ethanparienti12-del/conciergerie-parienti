Site prêt à uploader sur GitHub puis à déployer sur Vercel.

Fichiers à envoyer :
- index.html
- styles.css
- script.js

Étapes :
1. Dézippe le dossier.
2. Sur GitHub, clique sur "choose your files".
3. Sélectionne les 3 fichiers.
4. Clique sur "Commit changes".
5. Sur Vercel, importe le dépôt GitHub.
